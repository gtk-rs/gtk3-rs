Struct gtk::Bin

source · [−]
#[repr(transparent)]
pub struct Bin { /* private fields */ }
Expand description

The Bin widget is a container with just one child. It is not very useful itself, but it is useful for deriving subclasses, since it provides common code needed for handling a single child widget.

Many GTK+ widgets are subclasses of Bin, including Window, Button, Frame, GtkHandleBox or ScrolledWindow.

This is an Abstract Base Class, you cannot instantiate it.

Implements

BinExt, ContainerExt, WidgetExt, glib::ObjectExt, BuildableExt, ContainerExtManual, WidgetExtManual, BuildableExtManual
